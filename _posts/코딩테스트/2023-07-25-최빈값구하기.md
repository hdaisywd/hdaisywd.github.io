[최빈값 구하기 문제](https://school.programmers.co.kr/learn/courses/30/lessons/120812)를 풀어보았다. 아직 swift 문법이 부자연스러워 조금 헤맸다. 

```swift
import Foundation

func solution(_ array:[Int]) -> Int {
    var most: [Int:Int] = [:]
    for e in array {
        if most[e] == nil {
            most[e] = 0
        }
        else {
            most[e]! += 1
        }
    }
    
    var count = 0
    var ans = 0
    for (i, j) in most {
        if j == most.values.max() {
            count += 1
            ans = i
        }
    }
    
    return count == 1 ? ans : -1
}
```

배운것: array에서 key로 value를 찾을때도 optional값이 반환되므로 언래핑을 해줘야한다.
