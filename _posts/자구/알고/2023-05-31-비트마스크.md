## 비스마스크(Bit Mask)란?

컴퓨터는 모든 데이터를 이진수로 표현을 한다는 것은 익히 알고 있을 것이다. 그리고 이러한 최소 단위의 데이터를 저장하고 있는 공간을 비트(Bit)라고 하는데, 이 공간은 1과 0으로 표현이 된다. (따라서 이진수!) 

이러한 비트의 특성을 이용해서 이진수 표현을 자료구조로 쓰는 기법을 마트마스킹이라고 한다. 백준 문제를 풀다보면 분류에서 비트마스킹을 종종 마주칠 수 있다. 집합 문제를 풀어야 하는데 제한 시간이 짧을 때 사용해주면 손쉽게 통과할 수 있다. 
*(사담.. 처음 코딩 공부할 때 비트 연산자를 보고 이걸 언제 쓴다는 거야? 하고 콧방귀를 쳤는데 생각보다 쓸일이 많다!)*

**비트마스크의 장점**

1. 수행 시간의 단축

비트마스크는 bit 연산이기 때문에 시간 복잡도가 O(1)인 경우가 많다. 따라서 다른 자료구조보다 빠르게 작동한다. 

2. 코드의 단축

비트마스크를 집한 연산들을 한줄로 표현이 가능하다. 따라서 코드가 보다 간결하게 쓰일 수 있다. 

3. 메모리 사용량 감소

비트마스크는 많은 수를 포현할 때 효율적이다. (Ex. 10개의 비트는 2^10가지의 경우를 나타낼 수 있다.)

|&|AND 연산자, 비교하는 비트가 둘 다 참일 때 만족|0|0|0|
|||0|1|0|
|\||OR 연산자, 비교하는 비트가 둘 중 하나라도 참이면 만족|0|0|0|
|||0|1|1|
|^|XOR 연산자, 비교하는 비트 둘 중 하나만 참일때 만족|0|0|0|
|||0|1|1|
|~|NOT 연산자, 보수 연산으로 1->0, 0->1|0||1|
|||1||0|

|>>|오른쪽 시프트 연산, 변수의 값을 지정된 값의 비트만큼 오른쪽으로 이동(/2)|0010>>1||0001|

## 비트마스크를 통한 집합 표현

비트마스크를 이용하면 집합을 쉽게 표현할 수 있다. 원소의 개수가 n개인 집합이 있다고 하면, 원소에 0부터 n-1까지 번호를 부여해서 부분 집합으로 나타낼 수 있다. 번호에 해당하는 비트 하나가 하나의 원소 상태며, 비트가 1이면 해당 원소가 집합에 포함되어 있다는 의미고, 0이면 포함되지 않는다는 의미이다. 

예를 들어 1부터 20까지 들어가있는 집합이 있다고 치자. 

> 파이썬의 bin() 함수 
> 10진수 숫자를 2진수로 바꿔주는 함수로, 2진수임을 나타내는 0b가 앞에 붙어 이진수 숫자가 출력이 된다.
> 참고해야 할 점은 파이썬은 이진수를 str 형으로 표현한다는 것이다.

#### (1) 집합 만들어보기

```python
a = bin((1 << 20) - 1)
print(bin(a))
# 결과: 0b11111111111111111111
```

위와 같이 표현을 하면 집합이 현재 1부터 20까지 꽉 차있음을 알 수 있다. 

#### (2) 원소 포함 여부 확인

이제 원소가 들어있는지 아닌지 확인 해보자. 이는 AND 연산을 이용해서 확인해 볼 수 있다. 만약 원소가 존재한다면 무조건 양수가, 원소가 존재하지 않는다면 0이 나온다. 이러한 특성을 이용해보자. 

```python
a = 1 << 19
# bin(a) = 0b10000000000000000000
b = 1 << 5
# bin(b) = 0b100000

if a&b:
    print("Yup")
else:
    print("Nope")
# Nope
```

6번째 원소는 존재하지 않음을 알 수 있다. 

#### (3) 원소 삭제 

1부터 20까지 꽉 차 있는 집합에서 6번째 원소를 삭제해보자. OR 연산을 이용해서 할 수 있는데, 둘 중 하나가 0이면 0, 1이면 1인 특성을 이용하면 할 수 있다. 

```python
a = (1 << 20) - 1
# bin(a) = 0b11111111111111111111
b = 1 << 5
# bin(b) = 0b100000
c = a & ~b

print(bin(c))
# 0b11111111111111011111
```

NOT 연산으로 b를 뒤집어주면 삭제하고 싶은 원소만 0이 된다. 이를 원본 집합과 AND 연산하면 원하는 부분만 삭제해줄 수 있다. 

#### (4) 원소 토글

토글이란 비트가 켜져 있으면 끄고, 꺼져있으면 켜는 것을 말한다. (다크모드, 라이트 모드 전환하는 버튼같은 것도 토글이라고 부른다!)

```python
a = (1 << 20) - 1
# bin(a) = 0b11111111111111111111
b = 1 << 5
# bin(b) = 0b100000

a = a ^ b
print(bin(a))
# 0b11111111111111011111

a = a ^ b
print(bin(a))
# 0b11111111111111111111
```

XOR 연산은 둘 중 하나만 참일 때 성립한다. 0일때 1이 들어가면 1이 나오고, 1일때 1이 들어가면 0이 나오기 때문에 토글 기능을 구현할 수 있다.
